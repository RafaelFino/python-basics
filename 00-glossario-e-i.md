## E

### else
Palavra-chave com dois usos: (1) em condicionais: `else:` define o que acontece quando nenhuma condição anterior é verdadeira ("senão"); (2) em try/except: `else:` define código que executa apenas quando NÃO ocorreu nenhum erro no bloco try.

 [W3Schools — Python If...Else](https://www.w3schools.com/python/python_conditions.asp) · [Python Try Except](https://www.w3schools.com/python/python_try_except.asp)

### Endpoint
Endereço (URL) específico de uma API que realiza uma operação. É como o número de um ramal telefônico — cada ramal conecta você a um departamento diferente.

 [W3Schools — Python Requests](https://www.w3schools.com/python/module_requests.asp)

### Escopo de Variável
Região do código onde uma variável existe e pode ser acessada. Variáveis dentro de uma função (escopo local) só existem ali. Variáveis fora de funções (escopo global) podem ser acessadas em todo o programa. É como objetos dentro de um quarto vs objetos na sala.

 [W3Schools — Python Scope](https://www.w3schools.com/python/python_scope.asp)

### Exceção (Exception)
Erro que acontece durante a execução do programa. Quando ocorre, o programa para — a menos que você use `try/except` para tratá-la. É como um obstáculo inesperado no caminho.

 [W3Schools — Python Try Except](https://www.w3schools.com/python/python_try_except.asp)

### Exponenciação
Operação de multiplicar um número por ele mesmo várias vezes. Em Python: `**`. Exemplo: `2 ** 3` = `8` (2 × 2 × 2). É como dobrar uma folha de papel — cada dobra multiplica a espessura.

 [W3Schools — Python Operators](https://www.w3schools.com/python/python_operators.asp)

### Expressão
Combinação de valores, variáveis e operadores que o Python avalia para produzir um resultado. Exemplo: `2 + 3` resulta em `5`. É como uma conta matemática que o computador resolve.

 [W3Schools — Python Operators](https://www.w3schools.com/python/python_operators.asp)

## F

### f-string
Forma de formatar strings colocando `f` antes das aspas e usando `{}` para inserir variáveis. É como preencher lacunas em um formulário — os `{}` são as lacunas e as variáveis são os valores.

 [W3Schools — Python String Formatting](https://www.w3schools.com/python/python_string_formatting.asp)

### FAQ (Perguntas Frequentes)
Seção de Perguntas e Respostas Frequentes presente ao final de cada módulo deste curso. Contém dúvidas comuns, incluindo perguntas que estudantes podem ter receio de fazer. FAQ vem do inglês "Frequently Asked Questions".

 [W3Schools — Python Tutorial](https://www.w3schools.com/python/)

### FastAPI
Framework web moderno e rápido para construção de APIs HTTP em Python. É como um kit de ferramentas pronto que facilita a criação de serviços web. Usado neste curso para evoluir o projeto CRUD.

 [Documentação FastAPI](https://fastapi.tiangolo.com/)

### finally
Palavra-chave usada com `try/except` que define um bloco de código que SEMPRE será executado, independentemente de ter ocorrido erro ou não. É como a etapa de "limpar a cozinha" que você faz depois de cozinhar — não importa se a receita deu certo ou errado, a cozinha precisa ser limpa.

 [W3Schools — Python Try Except](https://www.w3schools.com/python/python_try_except.asp)

### Float
Tipo de dado que representa números decimais (com ponto). Exemplos: `3.14`, `9.99`, `0.5`. O nome vem de "floating point" (ponto flutuante). É como os preços em uma loja: R$ 5,99 seria `5.99` em Python.

 [W3Schools — Python Numbers](https://www.w3schools.com/python/python_numbers.asp)

### For (loop for)
Estrutura de repetição que executa um bloco de código para cada item de uma coleção ou um número determinado de vezes. É como passar por cada item de uma lista de compras, um por um. Usa a palavra-chave `for` seguida de `in`.

 [W3Schools — Python For Loops](https://www.w3schools.com/python/python_for_loops.asp)

### from
Palavra-chave usada junto com `import` para importar partes específicas de um módulo. Exemplo: `from math import sqrt` importa apenas a função de raiz quadrada do módulo math, em vez de importar tudo.

 [W3Schools — Python Modules](https://www.w3schools.com/python/python_modules.asp)

### Framework
Conjunto de ferramentas, bibliotecas e convenções prontas que facilitam o desenvolvimento. É como um kit de construção com peças pré-fabricadas. FastAPI é um exemplo de framework web.

 [W3Schools — Python Introduction](https://www.w3schools.com/python/python_intro.asp)

### Função
Bloco de código reutilizável que realiza uma tarefa específica. Você define uma vez e pode usar quantas vezes quiser. É como uma receita que você escreve uma vez e pode seguir várias vezes.

 [W3Schools — Python Functions](https://www.w3schools.com/python/python_functions.asp)

## G

### GET
Método HTTP usado para solicitar/ler dados de um servidor. Quando você acessa uma página na internet, seu navegador faz uma requisição GET. Em APIs, GET é usado para consultar dados.

 [W3Schools — HTTP Methods](https://www.w3schools.com/tags/ref_httpmethods.asp)

### global
Palavra-chave do Python que declara que uma variável dentro de uma função deve ser tratada como global (acessível em todo o programa). Sem `global`, variáveis criadas dentro de funções são locais. Use com cuidado — em geral, é melhor passar valores como parâmetros.

 [W3Schools — Python Scope](https://www.w3schools.com/python/python_scope.asp)

## H

### HTTP (HyperText Transfer Protocol)
Protocolo de comunicação da internet para transferir dados entre cliente e servidor. É como o idioma que o carteiro (internet) usa para entregar e receber cartas (dados) entre casas (computadores).

 [W3Schools — HTTP](https://www.w3schools.com/whatis/whatis_http.asp)

## I

### IDE (Integrated Development Environment)
Ambiente de Desenvolvimento Integrado. Programa que reúne ferramentas para facilitar a programação: editor, terminal, depurador. O VSCode é um exemplo. É como uma bancada de trabalho completa.

 [W3Schools — Python Getting Started](https://www.w3schools.com/python/python_getstarted.asp)

### if / elif / else
Palavras-chave para criar estruturas condicionais. `if` = "se", `elif` = "senão se", `else` = "senão". Permitem que o programa tome decisões diferentes dependendo das condições.

 [W3Schools — Python If...Else](https://www.w3schools.com/python/python_conditions.asp)

### Imutável
Algo que não pode ser alterado depois de criado. Em Python, strings e tuplas são imutáveis. É como uma foto impressa: depois de impressa, você não pode mudar a imagem.

 [W3Schools — Python Tuples](https://www.w3schools.com/python/python_tuples.asp)

### import
Palavra-chave do Python usada para importar módulos (bibliotecas) para usar no seu código. Exemplo: `import json` importa o módulo json. Pode ser combinada com `from` e `as`. É como pegar um livro de receitas da estante para consultar.

 [W3Schools — Python Modules](https://www.w3schools.com/python/python_modules.asp)

### in
Palavra-chave com dois usos: (1) em loops `for item in lista` significa "para cada item na lista"; (2) como operador de pertencimento `"a" in "banana"` verifica se "a" está dentro de "banana" (retorna True/False).

 [W3Schools — Python For Loops](https://www.w3schools.com/python/python_for_loops.asp) · [Python Operators](https://www.w3schools.com/python/python_operators.asp)

### Incremento
Ação de aumentar o valor de uma variável, geralmente de 1 em 1. Em Python: `contador += 1`. É como subir um degrau de cada vez em uma escada.

 [W3Schools — Python For Loops](https://www.w3schools.com/python/python_for_loops.asp)

### Indentação
Espaçamento à esquerda de uma linha de código. Em Python, define quais linhas pertencem a um bloco (corpo de if, for, função). Padrão: 4 espaços. É como itens de uma lista recuados para mostrar que pertencem àquela lista.

 [W3Schools — Python Indentation](https://www.w3schools.com/python/python_syntax.asp)

### __init__
Método especial de uma classe executado automaticamente quando um novo objeto é criado. É onde você define os atributos iniciais. O nome vem de "initialize" (inicializar). É como preencher a ficha de cadastro de um novo produto.

 [W3Schools — Python Classes](https://www.w3schools.com/python/python_classes.asp)

### Input
Função do Python que lê dados digitados pelo usuário no terminal. Sempre retorna uma string, mesmo que o usuário digite um número. É como perguntar algo a alguém e esperar a resposta.

 [W3Schools — Python User Input](https://www.w3schools.com/python/python_user_input.asp)

### Int / Inteiro
Tipo de dado que representa números inteiros (sem decimais). Exemplos: `1`, `42`, `-7`, `0`. É como contar objetos: você tem 3 maçãs, não 3,5.

 [W3Schools — Python Numbers](https://www.w3schools.com/python/python_numbers.asp)

### Interpretador
Programa que lê e executa código linha por linha, sem compilar tudo antes. O Python usa um interpretador. É como um tradutor simultâneo que traduz cada frase na hora.

 [W3Schools — Python Introduction](https://www.w3schools.com/python/python_intro.asp)

### is
Operador de identidade do Python que verifica se duas variáveis apontam para o mesmo objeto na memória (não apenas se têm o mesmo valor). Exemplo: `x is None` verifica se x é exatamente None. Diferente de `==` que compara valores.

 [W3Schools — Python Operators](https://www.w3schools.com/python/python_operators.asp)

### Iteração
Cada repetição (volta) de um loop. Se um loop executa 5 vezes, ele faz 5 iterações. É como cada volta que você dá em uma pista de corrida.

 [W3Schools — Python For Loops](https://www.w3schools.com/python/python_for_loops.asp)
