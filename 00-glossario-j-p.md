## J

### JSON (JavaScript Object Notation)
Formato de texto leve para troca de dados entre sistemas. Usa pares de chave e valor, similar aos dicionários do Python. Amplamente usado em APIs e configurações. É como uma ficha padronizada que qualquer sistema consegue ler.

 [W3Schools — Python JSON](https://www.w3schools.com/python/python_json.asp) · [Documentação Python — json](https://docs.python.org/pt-br/3/library/json.html)

### JVM (Java Virtual Machine)
Máquina Virtual Java. Ambiente que executa programas Java. O código é compilado para bytecode que a JVM interpreta. É como um tradutor universal que permite que o mesmo programa rode em diferentes computadores.

 [W3Schools — Java Introduction](https://www.w3schools.com/java/java_intro.asp)

## K

### kebab-case
Convenção de nomenclatura com palavras separadas por hífens em minúsculas. Exemplo: `meu-projeto`, `crud-produtos`. Em Python, usado para nomes de arquivos e pastas. Não pode ser usado para variáveis (o hífen é interpretado como subtração). Veja também: snake_case, CamelCase.

 [W3Schools — Python Variables](https://www.w3schools.com/python/python_variables.asp)

## L

### lambda
Palavra-chave para criar funções pequenas e anônimas (sem nome) em uma única linha. Exemplo: `dobro = lambda x: x * 2`. É como uma mini-receita rápida que você usa uma vez sem precisar dar um nome formal. Conceito avançado — não se preocupe com isso agora.

 [W3Schools — Python Lambda](https://www.w3schools.com/python/python_lambda.asp)

### Linguagem de Programação
Conjunto de regras e vocabulário que permite a humanos escrever instruções para o computador. Python, Java, C e JavaScript são exemplos. É como um idioma para "conversar" com o computador.

 [W3Schools — Python Introduction](https://www.w3schools.com/python/python_intro.asp)

### Lista (list)
Estrutura de dados que armazena uma sequência ordenada de elementos que podem ser alterados. Acessados por posição (índice). É como uma lista de compras onde você pode adicionar, remover e reorganizar itens.

 [W3Schools — Python Lists](https://www.w3schools.com/python/python_lists.asp)

### Loop
Estrutura que repete um bloco de código várias vezes. Python tem `for` (repete para cada item) e `while` (repete enquanto condição for verdadeira). É como dar voltas em uma pista.

 [W3Schools — Python While Loops](https://www.w3schools.com/python/python_while_loops.asp) · [Python For Loops](https://www.w3schools.com/python/python_for_loops.asp)

## M

### Markdown
Linguagem de formatação de texto simples usada para criar documentos com títulos, listas, links e blocos de código. Os arquivos deste curso (extensão `.md`) são escritos em Markdown.

 [W3Schools — HTML](https://www.w3schools.com/html/) — _Conceitos de formatação_

### Match/Case
Estrutura de controle do Python 3.10+ que compara um valor contra múltiplos padrões. Equivalente ao switch/case de outras linguagens. É como um cardápio onde você escolhe uma opção e o programa executa a ação correspondente.

 [Documentação Python — Match Statement](https://docs.python.org/pt-br/3/tutorial/controlflow.html#match-statements)

### Método
Função que pertence a um objeto ou classe. É uma ação que um objeto pode realizar. Exemplo: uma lista tem o método `.append()` para adicionar itens.

 [W3Schools — Python Classes](https://www.w3schools.com/python/python_classes.asp)

### Middleware
Código que executa antes ou depois de cada requisição HTTP em uma API. Por exemplo, um middleware de log registra todas as requisições, e um middleware de CORS adiciona headers de segurança. É como um porteiro que verifica cada pessoa que entra e sai de um prédio.

 [Documentação FastAPI — Middleware](https://fastapi.tiangolo.com/)

### Modelagem de Dados
Processo de planejar a estrutura dos dados antes de implementá-los. Envolve decidir quais campos uma entidade deve ter e qual tipo de dado cada campo usa. É como planejar uma ficha cadastral antes de preencher.

 [W3Schools — Python Dictionaries](https://www.w3schools.com/python/python_dictionaries.asp)

### Módulo (em Python)
Arquivo Python (.py) que contém código reutilizável (funções, classes, variáveis). Pode ser importado com `import`. É como um livro de receitas que você consulta quando precisa.

 [W3Schools — Python Modules](https://www.w3schools.com/python/python_modules.asp)

### Módulo (operação matemática)
Operação que retorna o resto de uma divisão. Em Python: `%`. Exemplo: `10 % 3` = `1`, porque 10 ÷ 3 = 3 com resto 1. É como dividir 10 balas entre 3 amigos — cada um fica com 3 e sobra 1.

 [W3Schools — Python Operators](https://www.w3schools.com/python/python_operators.asp)

## N

### .NET
Plataforma de desenvolvimento da Microsoft que inclui linguagens como C# e VB.NET. Programas .NET são compilados para um formato intermediário e executados por uma máquina virtual (CLR), similar à JVM do Java.

 [W3Schools — C# Introduction](https://www.w3schools.com/cs/cs_intro.php)

### None
Valor especial do Python que representa "nada" ou "vazio". Uma função sem `return` retorna `None`. É como uma caixa vazia — ela existe, mas não tem nada dentro.

 [W3Schools — Python None](https://www.w3schools.com/python/ref_keyword_none.asp)

## O

### Objeto
Instância criada a partir de uma classe. Se a classe é o molde (receita), o objeto é o produto final (bolo). Cada objeto tem seus próprios valores de atributos.

 [W3Schools — Python Classes](https://www.w3schools.com/python/python_classes.asp)

### OpenAPI
Especificação (padrão) que descreve a estrutura de uma API em formato JSON. O FastAPI gera a especificação OpenAPI automaticamente a partir do código Python. O Swagger UI lê essa especificação e gera a interface gráfica de documentação.

 [Documentação FastAPI](https://fastapi.tiangolo.com/)

### Operador
Símbolo que realiza uma operação sobre valores. Existem operadores matemáticos (`+`, `-`, `*`, `/`), de comparação (`==`, `!=`, `>`, `<`) e lógicos (`and`, `or`, `not`).

 [W3Schools — Python Operators](https://www.w3schools.com/python/python_operators.asp)

### Operando
Valor sobre o qual um operador atua. Na expressão `5 + 3`, os operandos são `5` e `3`, e o operador é `+`. É como os ingredientes de uma receita — o operador é a ação e os operandos são os ingredientes.

 [W3Schools — Python Operators](https://www.w3schools.com/python/python_operators.asp)

## P

### Pacote (Package)
Coleção de módulos Python organizados em uma pasta com um arquivo `__init__.py`. Pacotes permitem organizar código em subpastas. É como uma estante com várias prateleiras (módulos), cada uma com livros (funções e classes) sobre um assunto.

 [W3Schools — Python Modules](https://www.w3schools.com/python/python_modules.asp)

### Parâmetro
Variável definida na declaração de uma função que receberá um valor quando a função for chamada. É como o espaço reservado em um formulário — o parâmetro é o campo, e o argumento é o valor que preenche. Veja também: Argumento.

 [W3Schools — Python Functions](https://www.w3schools.com/python/python_functions.asp)

### pass
Palavra-chave do Python que não faz nada — serve como placeholder (espaço reservado) quando você precisa de um bloco de código mas ainda não escreveu o conteúdo. Exemplo: `def minha_funcao(): pass` cria uma função vazia que não faz nada (ainda). É como reservar um espaço em branco para preencher depois.

 [W3Schools — Python Functions](https://www.w3schools.com/python/python_functions.asp)

### PEP 8
Guia de estilo oficial do Python que define convenções de formatação: espaçamento, nomenclatura, comprimento de linhas. Seguir o PEP 8 torna o código mais legível e profissional.

 [PEP 8 — Style Guide](https://peps.python.org/pep-0008/)

### Pip
Gerenciador de pacotes do Python. Permite instalar, atualizar e remover bibliotecas externas. É como uma loja de aplicativos para o Python.

 [W3Schools — Python PIP](https://www.w3schools.com/python/python_pip.asp)

### POST
Método HTTP usado para enviar dados ao servidor para criar um novo recurso. Em APIs, POST é usado para adicionar novos registros (como cadastrar um novo produto).

 [W3Schools — HTTP Methods](https://www.w3schools.com/tags/ref_httpmethods.asp)

### Postman
Ferramenta popular para testar APIs HTTP. Permite enviar requisições GET, POST, PUT e DELETE com interface gráfica. Pode importar a especificação OpenAPI para criar requisições automaticamente. É uma alternativa ao curl e ao Swagger UI.

 [W3Schools — HTTP Methods](https://www.w3schools.com/tags/ref_httpmethods.asp)

### Print
Função do Python que exibe informações no terminal. É como falar em voz alta — o programa "diz" algo para você ver na tela.

 [W3Schools — Python Print](https://www.w3schools.com/python/ref_func_print.asp)

### Proposta de Teste
Conjunto de exemplos de entrada e saída esperada que acompanham cada exercício deste curso. Permite executar o programa com entradas específicas e verificar manualmente se a saída corresponde à esperada. É como conferir se o bolo ficou igual à foto da receita.

 [W3Schools — Python Tutorial](https://www.w3schools.com/python/)

### PUT
Método HTTP usado para atualizar um recurso existente no servidor. Em APIs, PUT é usado para editar registros (como alterar o preço de um produto).

 [W3Schools — HTTP Methods](https://www.w3schools.com/tags/ref_httpmethods.asp)

### Pydantic
Biblioteca Python usada pelo FastAPI para validação de dados. Define modelos de dados com tipos e regras de validação automática. É como um formulário inteligente que verifica se os dados preenchidos estão corretos.

 [Documentação FastAPI](https://fastapi.tiangolo.com/)

### Python
Linguagem de programação interpretada, de alto nível, criada por Guido van Rossum em 1991. Conhecida por sua sintaxe limpa e legível, é uma das linguagens mais populares do mundo.

 [W3Schools — Python Tutorial](https://www.w3schools.com/python/) · [Documentação Oficial](https://docs.python.org/pt-br/3/)
