# Glossário do Curso

Bem-vindo ao glossário! 

Este é o seu dicionário de consulta rápida para todos os termos técnicos e comandos Python usados ao longo do curso. Sempre que encontrar uma palavra ou conceito que não conhece, volte aqui para conferir o significado.

---

## Como Usar Este Glossário

- Os termos estão organizados em **ordem alfabética**, divididos em arquivos por faixas de letras para facilitar a navegação
- Cada termo tem uma **explicação simples** em português, pensada para quem nunca programou
- Cada termo inclui **links para fontes externas** onde você pode se aprofundar
- Você pode consultar este glossário **quantas vezes precisar** — não há limite!
- Se um termo não estiver aqui, procure no módulo onde ele apareceu ou pergunte ao seu professor

> **Dica:** use o atalho `Ctrl + F` no navegador ou editor para buscar um termo específico dentro de cada arquivo.
