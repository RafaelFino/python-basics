## R

### raise
Palavra-chave do Python usada para levantar (provocar) uma exceção intencionalmente. Exemplo: `raise ValueError("Preço inválido")` faz o programa gerar um erro de valor com a mensagem especificada. Útil para validar dados de entrada.

 [W3Schools — Python Try Except](https://www.w3schools.com/python/python_try_except.asp)

### Range
Função do Python que gera uma sequência de números. Muito usada com loops `for`. `range(5)` gera 0, 1, 2, 3, 4. É como numerar os degraus de uma escada.

 [W3Schools — Python Range](https://www.w3schools.com/python/ref_func_range.asp)

### README
Arquivo principal de um projeto que serve como ponto de entrada e guia de orientação. O nome vem do inglês "read me" (leia-me). Neste curso, contém o índice de módulos, guia de estudos e instruções de uso.

 [W3Schools — Python Getting Started](https://www.w3schools.com/python/python_getstarted.asp)

### Requirements.txt
Arquivo que lista todas as bibliotecas externas que um projeto Python precisa, com suas versões. É como a lista de ingredientes de uma receita.

 [W3Schools — Python PIP](https://www.w3schools.com/python/python_pip.asp)

### Resposta Comentada
Solução completa de um exercício com comentários detalhados em português em cada linha do código. Presente em todos os exercícios deste curso como referência de aprendizado — mas sempre tente resolver sozinho primeiro!

 [W3Schools — Python Comments](https://www.w3schools.com/python/python_comments.asp)

### Return
Palavra-chave dentro de uma função para devolver um valor como resultado. É como a resposta que uma máquina dá depois de processar algo.

 [W3Schools — Python Functions](https://www.w3schools.com/python/python_functions.asp)

## S

### Script
Programa escrito em linguagem interpretada (como Python) executado linha por linha, sem compilação prévia. É como ler e seguir instruções uma por uma.

 [W3Schools — Python Introduction](https://www.w3schools.com/python/python_intro.asp)

### Self
Parâmetro especial em métodos de classes Python que se refere ao próprio objeto. É como dizer "eu mesmo" — quando um objeto chama um método, `self` é a forma do método saber qual objeto está sendo usado.

 [W3Schools — Python Classes](https://www.w3schools.com/python/python_classes.asp)

### Slicing / Fatiamento
Técnica para extrair uma parte de uma sequência (string, lista, tupla) usando índices. `texto[0:5]` pega os 5 primeiros caracteres. É como cortar uma fatia de bolo.

 [W3Schools — Python Slicing Strings](https://www.w3schools.com/python/python_strings_slicing.asp)

### snake_case
Convenção de nomenclatura com palavras separadas por underscores em minúsculas. Exemplo: `minha_variavel`, `calcular_total`. Em Python, usado para variáveis, funções e módulos. Veja também: CamelCase, kebab-case.

 [W3Schools — Python Variables](https://www.w3schools.com/python/python_variables.asp)

### SQLite
Banco de dados relacional leve que armazena dados em um único arquivo, sem servidor. É como um caderno de anotações que guarda tudo em um só lugar. Ideal para aprendizado.

 [Documentação Python — sqlite3](https://docs.python.org/pt-br/3/library/sqlite3.html)

### String (str)
Tipo de dado que representa texto. Criadas com aspas simples `'texto'` ou duplas `"texto"`. É como uma palavra ou frase escrita em um papel.

 [W3Schools — Python Strings](https://www.w3schools.com/python/python_strings.asp)

### Swagger
Ferramenta de documentação interativa de APIs que permite visualizar e testar endpoints no navegador. O FastAPI gera Swagger automaticamente. É como um catálogo interativo dos serviços de uma API.

 [Documentação FastAPI — Swagger](https://fastapi.tiangolo.com/tutorial/first-steps/)

## T

### Terminal
Programa que permite interagir com o computador digitando comandos de texto. No Linux, é a principal forma de executar programas e gerenciar arquivos. É como uma janela de conversa com o computador.

 [W3Schools — Python Getting Started](https://www.w3schools.com/python/python_getstarted.asp)

### True / False
Os dois valores do tipo booleano em Python. `True` significa verdadeiro e `False` significa falso. São usados em condições, comparações e operadores lógicos. Atenção: a primeira letra é sempre maiúscula (`True`, não `true`).

 [W3Schools — Python Booleans](https://www.w3schools.com/python/python_booleans.asp)

### Try/Except
Estrutura do Python para tratar erros em tempo de execução. O código em `try` é executado normalmente; se ocorrer erro, `except` é executado. É como ter um plano B: "tente fazer isso; se der errado, faça aquilo".

 [W3Schools — Python Try Except](https://www.w3schools.com/python/python_try_except.asp)

### Tupla (tuple)
Estrutura de dados imutável que armazena uma sequência ordenada de elementos entre parênteses. Similar a uma lista, mas não pode ser alterada após criação. É como uma data de nascimento — uma vez definida, não muda. Útil para dados fixos e retorno múltiplo de funções.

 [W3Schools — Python Tuples](https://www.w3schools.com/python/python_tuples.asp)

### Type / Tipo de Dado
Classificação que define que tipo de valor uma variável armazena. Tipos básicos: `int` (inteiro), `float` (decimal), `str` (texto), `bool` (verdadeiro/falso). A função `type()` mostra o tipo.

 [W3Schools — Python Data Types](https://www.w3schools.com/python/python_datatypes.asp)

## U

### URL (Uniform Resource Locator)
Endereço usado para acessar recursos na internet. Exemplo: `https://www.google.com`. Em APIs, cada endpoint tem uma URL específica.

 [W3Schools — HTML URLs](https://www.w3schools.com/html/html_urlencode.asp)

### Uvicorn
Servidor web usado para executar aplicações FastAPI. É o programa que "liga" sua API e a torna acessível via HTTP. É como ligar o fogão antes de cozinhar.

 [Documentação FastAPI](https://fastapi.tiangolo.com/)

## V

### Variável
Espaço na memória do computador que armazena um valor e tem um nome. É como uma caixa etiquetada: a etiqueta é o nome e o conteúdo é o valor. Você pode trocar o conteúdo a qualquer momento.

 [W3Schools — Python Variables](https://www.w3schools.com/python/python_variables.asp)

### Venv (Virtual Environment)
Ambiente virtual do Python que isola as dependências de um projeto. Cada projeto pode ter suas próprias bibliotecas sem interferir em outros. É como ter uma cozinha separada para cada receita.

 [Documentação Python — venv](https://docs.python.org/pt-br/3/library/venv.html)

### VSCode (Visual Studio Code)
Editor de código gratuito e popular da Microsoft. Usado neste curso como ambiente de desenvolvimento. Possui extensões para destaque de sintaxe, autocompletar e depurador.

 [W3Schools — Python Getting Started](https://www.w3schools.com/python/python_getstarted.asp)

## W

### While (loop while)
Estrutura de repetição que executa um bloco enquanto uma condição for verdadeira. É como mexer uma panela no fogo: "enquanto não estiver no ponto, continue mexendo".

 [W3Schools — Python While Loops](https://www.w3schools.com/python/python_while_loops.asp)

### with
Palavra-chave do Python usada como gerenciador de contexto, que garante que recursos sejam abertos e fechados corretamente. Muito usada para trabalhar com arquivos: `with open("dados.txt") as arquivo:` abre o arquivo e garante que ele será fechado automaticamente ao final do bloco. É como abrir uma torneira e ter certeza de que ela será fechada quando você terminar.

 [W3Schools — Python File Handling](https://www.w3schools.com/python/python_file_handling.asp)
